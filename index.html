HOW TO PLAY
Basics
Drag / TapSelect a block from your hand, then tap a cell to place it
HoldStore one block to use later — tap hold slot to swap
RerollDiscard your hand and draw 3 new blocks (costs 1 ⬡)
HammerRemove one already-placed cell (2 uses per game)
UndoTake back your last move (3 uses per game)
Clearing
Fill a complete row, column, or 3×3 box with no duplicate numbers to clear it and earn bonus points + ⬡ tokens.

If a row/col/box fills up with duplicates, it won't clear — the conflicting cells turn red. Use Hammer or place carefully to fix it.
Scoring
Place block+10 pts per cell × multiplier
Clear row/col+80 pts + 1 ⬡
Clear box+120 pts + 2 ⬡
ComboClear 3 times in a row → multiplier +0.5
Zoo Mode 🐾
Switch between numbers and animals using the 🐾 ZOO toggle in the top bar. Same rules apply — each animal represents a number 1–9.
